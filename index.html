PrepBench

Evaluating Large Language Models on NL-Driven Data Preparation

About PrepBench

PrepBench is a benchmark for natural-language-driven data preparation, evaluating how well systems can transform raw, messy tables into analysis-ready data from natural language requests.

Built on real-world Preppin’ Data challenges, PrepBench captures realistic, multi-step preparation workflows with ambiguous intents and irregular data. It focuses on three key capabilities—interactive disambiguation, robust prep-code generation, and code-to-workflow translation—to measure how close today’s LLM-based agents are to practical, user-verifiable data preparation.
